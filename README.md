## Install

For installing and using this colection in Icestudio follow these steps:

* Download the collection: [stable](https://github.com/FPGAwars/iceWires/archive/refs/tags/v0.1.0.zip) or [development](https://github.com/FPGAwars/iceWires/archive/refs/heads/main.zip)
* Install the collection: *Tools > Collections > Add*
* Select the collection: *Select > Collection*
